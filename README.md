# 64bitmul.bf
64bit精度の掛け算ができるbfです  
機械的な生成です  
深夜テンションで書いてコード汚いのでご了承ください

## 要求環境
8bit wrappingセル、テープ長40以上

## 使い方
バイナリでこんな感じのを標準入出力で入れたら動きます
```
01 EF CD AB 89 67 45 23 01
00 
```
初期値は1で、バイナリで`01 [8バイトの数値]`を入力してくと累積的に掛けます  
`00`で終了です  
入れる数値はリトルエンディアンです

## 使用用途
ベンチマークとかに使うといいと思います  
安定性は良くわからんのでテストに入れたい時はよく考えてほしいです
